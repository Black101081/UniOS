# UniOS

Hệ điều hành đa nền tảng với tích hợp AI và khả năng kết nối liền mạch.

## Mục tiêu
- Hỗ trợ đa nền tảng (PC và Mobile)
- Tích hợp AI để quản lý tài nguyên thông minh
- Kết nối liền mạch giữa các thiết bị

## Cấu trúc dự án
```
UniOS/
├── kernel/     # Nhân hệ điều hành
├── api/        # API interfaces
├── platforms/  # Mã nguồn đặc thù cho từng nền tảng
└── services/   # Các dịch vụ hệ thống
```
