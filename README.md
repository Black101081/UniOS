# UniOS

## 🌟 Tổng quan
UniOS là một hệ điều hành đa nền tảng được thiết kế để hỗ trợ:
- Máy tính cá nhân (PC).
- Thiết bị di động.
- Các thiết bị IoT.

Mục tiêu của UniOS là tối ưu hiệu năng, đảm bảo bảo mật cao, và cung cấp khả năng tương thích mạnh mẽ với các ứng dụng hiện tại.

---

## ⚙️ Tính năng chính
1. **Đa nền tảng:**
   - Hỗ trợ cả kiến trúc x86, x64 và ARM.
   - Tương thích với ứng dụng Windows, Linux, Android.
2. **Bảo mật cao:**
   - Tích hợp sandboxing để cô lập ứng dụng.
   - Hỗ trợ mã hóa dữ liệu đầu cuối (end-to-end encryption).
3. **Hiệu năng tối ưu:**
   - Kernel nhẹ, tùy chỉnh từ Linux Kernel.
   - Tối ưu hóa tài nguyên cho từng loại thiết bị.
