# UniOS Kernel Documentation

## Tổng quan
Kernel của UniOS được thiết kế để quản lý:
- Bộ nhớ (Memory Management)
- Tiến trình (Process Management)
- Hệ thống tập tin (File System)

## Hướng dẫn phát triển
1. Chạy `kernel_init.c` để khởi động kernel.
2. Thêm các module bảo mật và tối ưu hiệu suất.
